This is my review of Dashlane Password Manager.

What is Dashlane

Dashlane is a Great Password manager with AES-256 encryption and a lot of Useful Features.

Features:

  • Encryption - AES-256 encryption
  • Autofill - Dashlane will suggest your Password, Username, and Personal info stored in Dashlane.
  • Password Generator - Generate secure password to 40 characters.
  • Password sharing
  • Password health - This feature will give you a score for your passwords and show how much strong, compromised, reused, weak passwords you have.
  • Password changer - This feature can auto change Your Passwords. (not all passwords support this feature and Dashlane Premium is required )
  • Dark web monitoring - You can add a maximum of 5 emails for scanning. Dashlane will find is any leaked information associated with your emails. (Dashlane Premium required )
  • VPN ( Dashlane Premium required )
  • Add personal info such as Phone Numbers, Emails, Websites, Names
  • Add IDs
  • Sync your passwords with another device ( Dashlane premium required )

    • If Your Using Free version of Dashlane:

  • Maximum 50 Passwords
  • 1 device only
  • No Dashlane VPN
  • No dark web monitoring
  • No auto password changer feature
